Подключение к магии слов

Ключ API (Google Gemini)

Чтобы сайт мог бесплатно писать для тебя фанфики, нужно подключить его к нейросети Google Gemini. Это занимает ровно одну минуту:

  1. Перейди на сайт Google AI Studio ↗ (понадобится Google-аккаунт).
  2. Нажми синюю кнопку "Create API key".
  3. Скопируй длинный код, который появится на экране.
  4. Вставь его в поле ниже и нажми "Сохранить".
