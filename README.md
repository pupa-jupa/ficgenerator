# ✦ FicGenerator (Генератор Фанфиков) ✦

![FicGenerator Banner](https://img.shields.io/badge/AI-Gemini_2.5-blueviolet?style=for-the-badge&logo=google-gemini)
![Tech Stack](https://img.shields.io/badge/Stack-HTML%20%7C%20CSS%20%7C%20JS-gold?style=for-the-badge)

**FicGenerator** — это элегантное и мощное веб-приложение для создания литературных историй и фанфиков с помощью нейросетей Google Gemini. Забудьте о «страхе чистого листа»: создавайте глубокие сюжеты, живых персонажей и целые миры в несколько кликов.

---

## ✨ Основные возможности

- **🪶 Умная генерация**: Используйте мощь Gemini 2.5 (Flash или Pro) для написания художественных текстов любого жанра.
- **🎭 Карточки персонажей**: Создавайте детальные профили героев: от характера (добрый/злой, интроверт/экстраверт) до внешности и роли в сюжете.
- **🖼️ Визуальное вдохновение**: Загружайте изображения-референсы, и ИИ учтет их атмосферу и детали при написании истории.
- **✨ Тонкая настройка атмосферы**: Управляйте тональностью текста (от мрачного ангста до уютного флаффа) и рейтингом (G, PG-13, R, NC-17).
- **📚 Личная библиотека**: Сохраняйте лучшие истории в браузере, создавайте для них автоматические саммари для публикации на Ficbook или AO3.
- **🎲 Генератор идей**: Встроенная база из 60+ уникальных сюжетных тропов для мгновенного вдохновения.
- **🎬 Читалка с комфортом**: Настраивайте размер шрифта, гарнитуру и переключайтесь на тёмную тему для ночного чтения.

---

## 🛠 Технологический стек

- **Frontend**: Чистый HTML5, CSS3 (Vanilla CSS) и оптимизированный JavaScript.
- **AI Engine**: Google Gemini API (модели `gemini-1.5-flash` и `gemini-1.5-pro`).
- **Storage**: LocalStorage для безопасного хранения вашей библиотеки и API-ключа в браузере.
- **PDF**: Встроенная функция генерации PDF-версий ваших историй.

## 🚀 Быстрый старт

### 1. Подготовка

Для работы приложения не требуется установка тяжелых зависимостей. Просто скачайте проект и откройте `index.html` в любом современном браузере.

### 2. Настройка AI (Gemini)

Для «магии» слов вам понадобится бесплатный API-ключ:

1.  Перейдите в [Google AI Studio](https://aistudio.google.com/app/apikey).
2.  Нажмите **"Create API key"**.
3.  Скопируйте ключ и вставьте его во вкладке **«Настройки»** внутри приложения.
4.  Ваш ключ хранится локально в вашем браузере и не передается третьим лицам.

## 📖 Как получить лучший результат

Чтобы ваши истории были максимально качественными:

1.  **Детализируйте персонажей**: Чем больше вы напишете в «Особенностях характера», тем меньше ИИ будет «галлюцинировать» и тем точнее попадет в образ.
2.  **Используйте референсы**: Картинка с мрачным лесом действительно добавит в текст описания тумана и таинственных шорохов.
3.  **Экспериментируйте с тональностью**: Слайдер «Драма/Флафф» значительно меняет используемую лексику и темп повествования.
4.  **Примечания автора**: Здесь можно указать стиль (например, «пиши в стиле классического нуара» или «используй много метафор»).
