# ✦ FicGenerator (Генератор Фанфиков) ✦

![FicGenerator Banner](https://img.shields.io/badge/AI-Gemini_2.5-blueviolet?style=for-the-badge&logo=google-gemini)
![Tech Stack](https://img.shields.io/badge/Stack-HTML%20%7C%20CSS%20%7C%20JS-gold?style=for-the-badge)

**FicGenerator** — это элегантное и мощное веб-приложение для создания литературных историй и фанфиков с помощью нейросетей Google Gemini. Забудьте о «страхе чистого листа»: создавайте глубокие сюжеты, живых персонажей и целые миры в несколько кликов.

---

## ✨ Основные возможности

- **🪶 Умная генерация**: Используйте мощь Gemini 2.5 (Flash или Flash Lite) для написания художественных текстов любого жанра. Движок использует продвинутые паттерны (XML-схемы и Chain-of-Thought) для создания глубоких сюжетов с правильным темпом.
- **🎭 Карточки персонажей**: Создавайте детальные профили героев с визуальным выбором архетипов.
- **🖼️ Визуальное вдохновение**: Загружайте изображения-референсы, и ИИ учтет их атмосферу и детали при написании истории.
- **📚 Личная библиотека**: Сохраняйте лучшие истории локально в браузере в премиальном дизайне «Dark Academia Magic» с поддержкой жанровых эмодзи.
- **🎲 Генератор идей**: Интегрированная рандомизация фандомов, пэйрингов и 60+ уникальных сюжетных тропов.
- **🎬 Эстетика и Комфорт**: Полностью погружающая «Тёмная Академия» (Dark Mode) и умная читалка с настройкой шрифта.
- **✏️ Интерактивное редактирование**: Продолжайте тексты, генерируйте повороты сюжета кнопками или переписывайте текст прямо в браузере (`contenteditable`).
- **🖨️ PDF Экспорт**: Умная генерация PDF через системную печать с чистым, отформатированным стилем (со скрытием всего UI).

---

## 🛠 Технологический стек

- **Frontend**: Чистый HTML5, CSS3 (Vanilla CSS с CSS-переменными) и оптимизированный JavaScript.
- **AI Engine**: Google Gemini API (модели `gemini-2.0-flash` и `gemini-2.5-flash-lite`), форматирование Markdown + DOMPurify.
- **Storage**: API `localforage` для надежного асинхронного хранения библиотеки историй.

## 🚀 Быстрый старт

### 1. Подготовка

Для работы приложения не требуется установка тяжелых зависимостей. Просто скачайте проект и откройте `index.html` в любом современном браузере.

### 2. Настройка AI (Gemini)

Для «магии» слов вам понадобится бесплатный API-ключ:

1. Перейдите в [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Нажмите **"Create API key"**.
3. Скопируйте ключ и вставьте его во вкладке **«Настройки»** внутри приложения.
4. Ваш ключ хранится локально в вашем браузере и не передается третьим лицам.

## 📖 Как получить лучший результат

Чтобы ваши истории были максимально качественными:

1. **Детализируйте персонажей**: Чем точнее вы выберете архетип и опишете особенности героя, тем ярче он получится.
2. **Используйте референсы**: Картинка с мрачным лесом действительно добавит в текст описания тумана и таинственных шорохов.
3. **Используйте теги и канон**: Если вы пишете фанфик (например, Гарри Поттер), движок автоматически включает жесткие рамки следования канону, чтобы не выдумывать лишнего лора, если вы сами того не попросите.
4. **Примечания автора (Свой промпт)**: Здесь можно напрямую обращаться к ИИ (например, «пиши в стиле детективного нуара» или «акцентируй внимание на эмоциях»).
